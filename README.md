# Glossina Transcriptome-Based AMP Discovery

## Project Overview

Computational identification and characterization of antimicrobial peptides (AMPs) from transcriptomic data of *Glossina morsitans morsitans* (tsetse fly).

## Research Objective

To identify putative antimicrobial peptides and evaluate their physicochemical properties, predicted safety profiles, and structural characteristics using an integrated bioinformatics pipeline.

## Methods

### Data Processing

* RNA-seq preprocessing (Fastp)
* Transcriptome assembly (Trinity)
* Transcript quantification (Salmon)

### AMP Identification

* Homology-based AMP identification (BLASTx)
* Antimicrobial Peptide Database (APD3) screening

### Peptide Characterization

* Physicochemical characterization
* Net charge analysis
* Hydrophobicity analysis
* GRAVY index calculation

### Predicted Safety Assessment

* Hemolyticity prediction (HemoPI)
* Toxicity prediction (ToxinPred)
* Allergenicity prediction (AllerTOP)

### Structural Analysis

* Structural modeling (PEP-FOLD3)
* Structure visualization (PyMOL)

## Key Findings

* Identified multiple putative AMP candidates from the tsetse fly transcriptome.
* Characterized predicted physicochemical properties relevant to antimicrobial activity.
* Evaluated predicted safety profiles (hemolyticity, toxicity, and allergenicity) using computational tools.
* Generated predicted structural models for selected AMP candidates.
